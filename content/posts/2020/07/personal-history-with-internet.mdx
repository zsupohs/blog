　2020年もそろそろ終わりに近づいてきたところで、私のインターネット史について語ろうじゃないか。みんなはインターネットといつから付き合い始めてる？昭和から？平成から？令和から？いずれにせよ、あなたはおそらくインターネットから大きな影響を受けたであろう。実は……私もその一人だ。

　私がインターネットに初めて触れたのはたぶん、2001年頃だったと思う。当時若く偉大であった父が Personal Computer を購入し（値段は知らないけど、たぶん20万円くらいじゃなかったかな……VAIOのやつです）、父が使っていない時には遊ばせて貰えた。これが最初の体験だった。

　最初はパソコンに入ってた 3D ピンボールゲーム（アレね）をやったり、ペイントソフトでお絵描きをしたりしてとても楽しんでいた（マインスイーパーはアホな小学生の頭ではあまり理解できなかったらしく遊んだ記憶がない）。私は小学生の頃から友達が一人もいなかったものだから、それ以前はよくスケッチブックに自作のキャラクターが活躍する四コマ漫画やイラストを描き家族に見せたものだが、パソコン以後はそれがＤＥＪＩＴＡＲＵの力でキレイな色！まっすぐな線！になった。それにとても感動したものだった。プリンターで印刷してマジマジと見てたあの頃の絵はもう全部捨てられたのかな。

　その後、インターネットを閲覧するようになると、大体私のインターネッツ生息地域は無料ゲーム集？や面白フラッシュ倉庫？みたいなところに大体限定されていたが、当時遊んでいたテレビゲーム（三國無双とか）の情報を蒐集するためにゲームの攻略サイトを見るようになり、それから更に領域を広げて三国志系の情報サイトや個人サイトなどに足を運ぶようになった。ざっと思い出す限り、以下のようなものがあったと思う。三国志連環（リンク集！相互リンク！直リン禁止！懐かしい響きだ……）とかあったしね。三國無双の陸遜のエッチなイラストからは潜在意識に変な影響を与えられたのかもしれない。

- [伊籍の館](http://iseki.gozaru.jp/)
- [夷陵賓館](https://hb5.seikyou.ne.jp/home/misemise/)
- [鍋島の野望](http://web.archive.org/web/20060822044847/nabeyabo.com/)（閉鎖されたようです）

特に私に大きな影響を及ぼしたのは以下のサイトだ。

- [度胸だめし](http://www.110kz.com/003/)

今思えば、度胸も何もしょうもないサイトだな～～って感じだけれど、当時は死ぬほどドキドキしたものだ。仕組みがわかると、じゃあどうやって作ってるんだよということが気になって、いろんなサイトを調べてどうやらこいつは JavaScript とかいうので動いてるらしい。そしたら HTML だか CSS とかいうのもあるらしい。なんだこれ。

　理由はもう忘れたが、強く興味を惹かれたのだと思う。[とほほのWWW入門](http://www.tohoho-web.com/)なんかも見てたけど、誰かが「とほほは正しくない！適当だ！仕様通りに書け！HTMLとCSSは云々～」とか言ってたので、自分で考えることの知らないスクリプトキディのバカガキの私も影響を受けて「とほほは正しくない！」みたいに気取りやがってた記憶がある。クソガキが。でもとほほのWWW入門は今でも続いてて、最近では Rust とかの入門もやっててすごすぎる。

　他には、スーパー正男（聞いたか？2020年にスーパー正男という単語だ……）を遊んで自分でもステージを作ってみようとして挫折したり、JavaScript でつくれる簡単ブロック崩し！みたいな解説サイトに挑戦して挫折したり……ただ、こうした一連の体験は、私のプログラミングに対しての関心を静かに涵養していったのだなあということにしとくと、なんか人に歴史ありっぽい。

　小学生の終わりころにはオカルト関係（そういえば最近 [Naokiman Show](https://www.youtube.com/channel/UC4lN5sizuJraSHqy99xTy6Q) とかいうユーチューバー見てる……クソくだらないと思ってたけど、全部アヌンナキのせいなんだよな～３・６・９の法則があってさ～）のサイトも見るようになり、とあるオカルト系サイトの掲示板に出入りしてたりしてた。みんなも一度はオカルトにハマるよね～、[超魔界帝国](http://daimaohgun.web.fc2.com/)なんかも当時読んでいた。あまりの記事のバカさに「この人たちほんとに大人か？何やってんだ？」と思ってたな～。宵闇の森とかいうバナーからしてクソ怖いサイトは全く入ることができなかった。当時はそういう（自分にとっての）禁足地みたいなのがあって、ワクワクしなかったか？キミたち。

　後はウイルス入れられたり、ブラクラ踏んだり、「すごい写真だよ！」って一言だけ書いてあるURL踏んだら裸の女性の写真でびっくりしてすぐ閉じてドキドキしたりしてた。

　[KAGAYA](http://www.kagayastudio.com/) を知ったのもそれくらいの時期だったと思う。ペイントソフトで単色塗りのカスみたいな絵を描いてた私にとって超絶美麗なイラストは衝撃的でずっとディスプレイ越しにマジマジ眺めていたと思う。当時とくに好きだったのは『セレスティアル・エクスプローリング』シリーズ全般と『銀河鉄道の夜』シリーズの『銀河へ続く道』だったと思う。中学生の頃美術部に所属してたんだけど、まるっきりKAGAYAを模倣した絵を描いてた、色鉛筆で。

　今でも結構思い出せるものですね、よく読んでたテキストサイトとかまだいろいろあるけど、疲れたんで終わり……。
